Assignment 10 Grader





Directions

Create a program that meets the following conditions:

  1. A method that takes in one parameter
  2. 1 Class and 1 Subclass. Each should include:
    1. 2 unique variables
    2. 1 unique method
    3. 1 overridden method
    4. 2 constructors
  3. An Array that is traversed with a for-each (advanced for) loop
  4. An ArrayList that is traversed with a standard for loop
  5. A 2D Array that is traversed however you like
  6. A recursive method
Bonus: Add a sorting method to get one of your arrays in order and use binary search to get the index of an item.
